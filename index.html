⚙️ Pengaturan API Key

Masukkan YouTube Data API v3 key kamu. Key hanya disimpan di memori untuk sesi ini (tidak pernah masuk ke LocalStorage) — kamu perlu memasukkannya lagi setiap kali halaman dibuka ulang.

Belum punya key? Buat di Google Cloud Console, aktifkan "YouTube Data API v3".
