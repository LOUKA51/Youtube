⌨️ Petunjuk Hotkeys

Enter — kirim teks di kotak ketik ke Layar Proyektor
T — sembunyikan Pop-up Teks di Layar Proyektor
P — toggle Play / Pause video
I — tampilkan / sembunyikan panduan ini

⚙️ Pengaturan API Key

Masukkan YouTube Data API v3 key kamu. Key disimpan hanya di browser ini (LocalStorage) dan tidak pernah dikirim ke server lain selain Google.

Belum punya key? Buat di Google Cloud Console, aktifkan "YouTube Data API v3".
